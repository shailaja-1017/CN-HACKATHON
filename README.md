Public WiFi Security Risk Analysis Report


Project Title: Public WiFi Security Risk Analyzer
Platform Used: Kali Linux
Tools & Technologies: Nmap, Tcpdump, Netstat, Python


Done By:
G SHAILAJA 2420030447 AND 
S LAHARI KRISHNA 2420090055


Objective
To analyze security risks in a public WiFi network by scanning devices, identifying open ports, monitoring traffic, and detecting vulnerabilities.


Methodology
1. Network Identification using ip a
2. Network Scanning using nmap
3. Packet Analysis using tcpdump
4. Connection Monitoring using netstat
5. Risk Analysis using Python script
6. Report Generation using nano

Analysis Parameters
• Number of Open Ports
• Presence of Unencrypted Traffic
• Unknown Devices in Network


Result
Based on the analysis, the network risk level is classified as LOW, MEDIUM, or HIGH depending on the input parameters.
